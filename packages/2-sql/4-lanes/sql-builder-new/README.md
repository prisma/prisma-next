# @prisma-next/sql-lane-sql-builder-new

Type-level SQL query builder for Prisma Next.

## Responsibilities

- Provides a type-safe SQL query builder DSL with compile-time type inference
- Supports capability-gated features based on contract capabilities

## Dependencies

- `@prisma-next/sql-contract` — SQL contract types

## Architecture

- **Domain:** SQL
- **Layer:** Lanes
- **Plane:** Runtime

## Status

This package is currently under development.
