# @prisma-next/contract-psl

**Status:** Scaffolded placeholder package (future)

This package will contain the PSL parser and IR for Prisma Next.

## Package Classification

- **Domain**: framework
- **Layer**: authoring
- **Plane**: migration

## Overview

This package is part of the authoring ring and will provide PSL (Prisma Schema Language) parsing and IR generation capabilities.

## Package Status

This package is currently a placeholder for future implementation. It was created during the package layering scaffolding phase to signal the intended structure.

## Responsibilities

- **PSL Parsing**: Will parse Prisma Schema Language (PSL) files into contract IR
- **IR Generation**: Will generate contract IR from parsed PSL schemas
- **Authoring Integration**: Will integrate with the contract authoring system to support PSL-based contract creation

## Dependencies

This package is currently a placeholder with no dependencies. When implemented, it will likely depend on:
- **`@prisma-next/contract`**: Core contract types
- **`@prisma-next/contract-authoring`**: Authoring builder core
