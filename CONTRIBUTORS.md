# Contributing to Prisma Next

Prisma Next is currently in active development and is **not open to external contributions**.

At this time, only [Prisma](https://prisma.io) employees and contractors may contribute to this repository. We are not accepting pull requests, issues, or feature requests from the community.

We plan to open contributions in the future. Watch this repository for updates.

## Questions?

If you have questions about Prisma Next, please reach out through official [Prisma](https://prisma.io) channels.
